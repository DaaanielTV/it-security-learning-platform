# 🛡️ IT Security Learning Platform

An interactive web platform for mastering IT security concepts, from beginner to expert level! 🎯

## 🌟 Features

- 📚 Comprehensive learning modules covering:
  - Network Security
  - Encryption
  - Penetration Testing
  - Malware Analysis
  - And more!

- 🎮 Interactive Learning
  - Virtual Labs
  - Hands-on Exercises
  - Real-world Scenarios
  - Progress Tracking

- 🏆 Gamification
  - Achievement Badges
  - Progress Points
  - Leaderboards
  - Certificates

- 🔧 Technical Stack
  - React + TypeScript
  - Tailwind CSS
  - Supabase
  - Vite

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/DaaanielTV/it-security-learning-platform
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

## 🛠️ Development

- Build project:
```bash
npm run build
```

- Run linter:
```bash
npm run lint
```
